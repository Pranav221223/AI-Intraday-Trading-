AI Intraday Trading is a **machine learning powered stock prediction system** designed to forecast short-term price movements for intraday trading.
The project analyzes historical stock data and uses trained deep learning models to generate predictions through a **Flask-based web dashboard**.

This project demonstrates how **Artificial Intelligence, Machine Learning, and Financial Data Analysis** can be combined to build an intelligent trading support system.

---

# 🚀 Features

✅ AI-based stock price prediction
✅ Intraday trading analysis
✅ Multiple trained ML models for different stocks
✅ Data preprocessing and feature scaling
✅ Web dashboard for predictions
✅ Login and user authentication system
✅ Flask backend with HTML/CSS frontend
✅ Real-time prediction interface

# ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/AI_Intraday_Trading.git
cd AI_Intraday_Trading
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
```

Activate environment

**Windows**

```bash
venv\Scripts\activate
```

**Mac / Linux**

```bash
source venv/bin/activate
```

---

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Usage

Run the Flask application

```bash
python app.py
```

Open your browser and go to

```
http://127.0.0.1:5000
```

You can now:

• Login / Register
• View stock predictions
• Analyze AI model output
